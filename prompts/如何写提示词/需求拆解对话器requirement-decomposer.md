# 定位

定位：输入侧的交互式增强器。

Prompt Generator 是单轮的——用户丢一句话，系统自动补足。但有些复杂需求，单轮补足的准确率不够高。这个工具走另一条路：用最少的结构化追问，快速锁定用户真正要什么，然后再产出最终方案。

为什么有效： 很多时候用户自己也没想清楚。与其让系统猜，不如用 3-5 个高命中率的追问把模糊空间压到最小。这个工具特别适合高复杂度任务（技术方案、产品设计、战略分析等）。

# 提示词

我会给你一个需求或目标。

不要直接回答，也不要直接生成 Prompt。你的任务是先用最少的问题帮我把需求锁定到可执行的精度。

你必须完成以下工作：
1. 用一句话复述你理解的核心需求
2. 识别当前描述中最大的模糊点（最多 3 个）
3. 针对每个模糊点，给出一个精准的追问（不要问开放式大问题，要给选项或范围）
4. 如果我的需求本身就足够清晰，直接告诉我"无需追问"并说明原因

规则：
- 追问总数不超过 5 个
- 每个追问必须附带 2-3 个最常见的选项，帮我快速决策
- 不要问"你的目标是什么"这类无效问题，要问具体的取舍和边界
- 追问结束后，输出一份锁定后的需求摘要
